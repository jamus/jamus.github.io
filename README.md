# jamus.co.uk

Personal portfolio site. Hosted via GitHub Pages at [jamus.co.uk](https://jamus.co.uk).

## Stack

- Plain HTML with [Tailwind CSS v4](https://tailwindcss.com/) for styling
- [Mona Sans](https://github.com/github/mona-sans) variable font
- [Ionicons](https://ionic.io/ionicons) for icons

## Development

```bash
npm install
npm run dev       # watch CSS + live-server
```

```bash
npm run build:css # minified CSS build
npm run format    # prettier
```
